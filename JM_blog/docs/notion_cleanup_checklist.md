## 2. 권한 점검

### 전 직원

- [ ] `진행중인 현장`에서 금액 관련 열 없음
- [ ] 고객명 없음
- [ ] 고객 연락처 없음
- [ ] 계약금액 없음
- [ ] 경비합계 없음
- [ ] 출금합계 없음
- [ ] 총지출 없음
- [ ] 재무 relation 없음

### 현장팀·관리팀

- [ ] `현장 경비/영수증` 접근 가능
- [ ] `출금 요청` 접근 가능
- [ ] 관리자 전용 재무/직원/고객 DB는 접근 불가

### 관리자

- [ ] 직원 명부 접근 가능
- [ ] 근태 기록 접근 가능
- [ ] 매출/매입 접근 가능
- [ ] 현장별 재무 접근 가능
- [ ] 고객 CRM 접근 가능
- [ ] 견적 현황 접근 가능
- [ ] 공사 아카이브 접근 가능

## 3. 모바일 보기 점검

이미 완료된 핵심 화면:

- [x] 진행중인 현장: 갤러리
- [x] 완료 현장: 갤러리
- [x] 공정 일정: 상태별 보드
- [x] 업무 현황: 진행상태별 보드
- [x] 현장 경비/영수증: 영수증 사진 갤러리
- [x] 출금 요청: 상태별 보드

- [ ] 자재/발주: 상태별 보드
- [ ] A/S: 접수상태별 보드
- [ ] 협력업체: 업종별 리스트
- [ ] 공지/매뉴얼: 중요도별 리스트
- [ ] 직원 명부: 리스트
- [ ] 근태 기록: 캘린더 또는 리스트

- [ ] 고객 CRM: 상담상태별 보드
- [ ] 견적 현황: 견적상태별 보드
- [ ] 공사 아카이브: 갤러리

## 4. 완공 현장 흐름

- [ ] 현장 단계가 `완공`이면 `현장 갤러리`에서 빠짐
- [ ] 현장 단계가 `완공`이면 `완료 현장`에 표시됨
- [ ] 완료 현장 표지는 `완성 사진`
- [ ] 자료는 이동하지 않고 같은 현장 카드 안에 유지
- [x] 신규 사진은 현장 카드의 `시공전 사진첩 / 시공중 사진첩 / 완성 사진첩`에서 관리
- [x] 잘못 올린 사진은 해당 사진첩 페이지에서 직원이 직접 삭제 가능
- [x] 전 직원 현장 DB의 기존 `시공전 사진` 속성 삭제 완료

## 5. 사용자에게 최종 안내할 내용

- 현장팀은 모바일에서 `JM 업무 시스템 → 메인 대시보드`만 열면 됨
- 진행 현장은 `현장 갤러리`
- 공정은 `공정 일정`
- 업무는 `업무 현황`
- 영수증은 `현장 경비/영수증`
- 출금은 `출금 요청`
- 돈 관련 정보는 관리자 영역에서만 확인
