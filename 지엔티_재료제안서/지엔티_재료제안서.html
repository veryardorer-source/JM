본 자료는 사무공간 인테리어 시공에 사용되는 주요 자재의 제조사·브랜드·재질·마감·색상 정보를 한 권으로 정리한 표준 레퍼런스입니다. 현장 여건과 예산에 따라 등급·시리즈는 유연하게 조정 가능합니다.

구성 1. 벽지 (Wallcovering) · 2. 알판 (Panel) · 3. 사무용가구 (Office Furniture) · 4. 붙박이가구 (Built-in Furniture)

· 모델명·시리즈: 대표 컬렉션 기준 (단종 시 동급 대체)

1벽지 (Wallcovering)

사무공간은 합지보다 내구성이 높은 실크벽지를 기본으로 적용하며, 친환경 등급(E0/SE0)을 우선 검토합니다.

주요 제조사 · 브랜드

제조사대표 브랜드 / 컬렉션특징
LX하우시스지인(Z:IN) 벽지 — 베스띠, 디아망, 엘레강스국내 점유율 1위. 실크·합지 풀라인업, 친환경 SE0 다수
개나리벽지모건(Morgan), 까사미아, 베스트실크벽지 강세, 패턴·솔리드 색상 폭 넓음
신한벽지아이비스(IVIS), 오엔, 까사보떼친환경(아이비스) 라인, 가성비 우수
디아이디(DID)DID 실크, 와이드광폭(106cm) 벽지로 이음매 최소화
제일벽지까사데코, 데코마인중가 라인 시공성 양호
수입오즈본앤리틀, 산데르손, 코로(KORO)포인트 월 / 임원실 등 디자인 강조 구간

재질 · 구조

합지벽지종이 2겹 합지. 단가 저렴, 통기성 우수. 사무실에는 회의실·창고 등 보조공간에 적용
실크벽지종이 베이스 + PVC 코팅. 오염·물걸레 청소 가능. 사무공간 표준
친환경 벽지SE0 / E0 등급. VOC·포름알데히드 저감. 신축·리모델링 권장
기능성 벽지방염(난연), 항균, 흡음 — 회의실·임원실·콜센터 등 용도별 선택

마감 · 텍스처

  • 무광 솔리드 — 사무공간 표준. 빛 반사 적어 모니터 작업 환경에 유리
  • 엠보싱 — 입체 질감, 벽면 미세 결함 커버력 우수
  • 패브릭 / 리넨 패턴 — 임원실·회의실 포인트
  • 스톤 / 콘크리트 패턴 — 모던 사무공간 / 라운지
  • 우드 그레인 — 따뜻한 톤 강조 공간

2알판 (Panel)

알판은 가구 도어·몸체·벽체 마감에 사용되는 판상 자재의 통칭입니다. 베이스 보드(PB/MDF) 위 마감재(PET, UV, 무늬목 등)를 입혀 사용합니다.

주요 제조사 · 브랜드

제조사주요 제품군특징
LX하우시스LPM·HPL·PET 강화도어, 보닥타일도어재·인테리어 필름 종합. 인지도 높음
한솔홈데코스타일우드, 알판 PB·MDF국내 PB 1위. 친환경 E0 등급
동화기업동화자연마루, MDF 알판마루재로 유명, 가구용 MDF 공급 활발
한화L&C칸스톤, 인테리어 필름엔지니어드 스톤·필름 강세
현대L&C보닥, 하이막스, 인테리어 필름하이그로시·솔리드 서피스
삼익산업HPL, LPM가구·인테리어용 고압라미네이트
해성DS / EGGER유럽산 PB·라미네이트수입 고급 라인, 색상·패턴 다양

베이스 보드 (코어)

PB (Particle Board)나무 입자 압착. 가구 몸체 표준. 두께 15·18·25mm. 단가 저렴
MDF (중밀도섬유판)섬유 압착. 표면 매끄러움 → 도장·UV 마감용. 두께 3·9·15·18mm
합판 (Plywood)박판 적층. 휨·강도 우수. 도어·구조재
친환경 등급SE0 (0.3mg/L 이하) / E0 (0.5mg/L 이하). 사무가구는 E0 이상 권장

마감재 (표면)

LPM (저압 멜라민)PB·MDF에 멜라민 시트 압착. 사무가구 표준. 내구성·가성비 우수
HPL (고압 라미네이트)고압 적층. 내스크래치·내열성 최상. 카운터·상판
PET고광택·무광 모두 가능. 색감 선명, 무늬목 표현 우수. 사무·주방 도어
UV 코팅고경도 광택 마감. 하이그로시 도어 대표
멤브레인PVC 시트 진공 압착. 곡면·몰딩 표현 가능
무늬목 / 천연 베니어천연 베니어 부착 후 도장. 임원실·고급 가구의 도어·전면 마감
인테리어 필름현장 부착형 PVC 필름(LX·현대 보닥). 리모델링 시 가구·문 재마감

3사무용가구 (Office Furniture)

책상·의자·수납장 중심. 국내 메이저 3사(퍼시스·코아스·한샘)와 디자인 의자 전문 시디즈가 사무공간 표준입니다.

주요 제조사 · 브랜드

제조사대표 라인업포지셔닝
퍼시스 (FURSYS)USIT, FX, 클라트, T70국내 사무가구 1위. 시스템 가구 표준. 대형 사옥 다수 납품
코아스 (KOAS)아폴로, 듀에토, 클라우드중대형 오피스. 가성비·디자인 균형
한샘 오피스샘스, 큐브, 메이트중소 오피스·스타트업 대응. 빠른 납기
시디즈 (SIDIZ)T50, T80, T90, 링고의자 전문. 사무용 의자 점유율 1위
일룸 (iloom)데스커(Desker), 린백1인용·홈오피스·스타트업
리바트 오피스워크온, 모디현대백화점 그룹. 중고가 라인
수입 (참고)허먼밀러, 스틸케이스, 비트라임원실·디자인 강조 공간. 에어론·립체어

책상 (Desk)

일반 책상1200 / 1400 / 1600 × 700mm 표준. 1인용 워크스테이션
시스템 책상모듈 조합형. ㄱ자·일자·맞대기. 와이어링 덕트 내장
모션 데스크 (전동 높이조절)650 ~ 1250mm 가변. 듀얼모터. 좌식·입식 전환. 시디즈 TX, 데스커 모션 등
회의 / 컨퍼런스 테이블2400~6000mm. 단판형·연결형. 콘센트 박스 매립
임원 책상1800~2400mm. 무늬목·도장 마감. 사이드 보조탁 포함

상판 · 프레임 재질

상판MFC(멜라민 코팅 PB) — 표준 / HPL — 고내구 / 무늬목 도장 — 임원실 / 강화유리 — 회의실
엣지 (모서리)PVC 엣지 1~3mm. ABS 엣지(친환경) / 포스트포밍(일체형) — 모서리 라운드
프레임스틸 분체도장(파우더코팅) — 표준 / 알루미늄 — 고급 / 우드 — 카페형 사무공간
서랍 / 캐비닛스틸 본체 + 분체도장 / LPM 마감 PB. 슬라이드: 풀오픈 볼베어링

4붙박이가구 (Built-in Furniture)

현장 치수 맞춤 제작. 사무공간에서는 탕비실 수납, 임원실 책장·장식장, 회의실 콘솔, 복도 수납장 등에 적용됩니다.

주요 제조사 · 시공 방식

유형대표 업체 / 브랜드특징
대기업 맞춤 가구한샘 키친바흐 / 한샘 우노, 리바트 키친, 에넥스본사 설계·제작·시공. AS 체계화. 단가 높음
중견 맞춤 가구넵스, 우아미, 까사미아 키친중상위 가격대. 임원실 / 라운지 적합
현장 제작인테리어 시공팀 직접 제작치수·디자인 자유도 최상. 현장 여건 100% 반영
주문 제작 (반맞춤)퍼니처바이, 마켓비, 인테리어 가구 공방온라인 견적·표준 사이즈 조합

내수합판탕비실 싱크대 하부장 등 수분 노출 구간

견적 / 자재 변동 안내

  • 본 자료의 제조사·모델·색상은 일반 표준 기준으로 작성되었으며, 실제 적용 자재는 현장 실측·예산·납기 협의 후 확정됩니다.
  • 친환경 등급(SE0/E0), KS·KCL 인증, 방염 인증 등은 요청 시 자재별 인증서를 첨부 드립니다.
  • 수입 자재·고급 라인은 단가 변동 폭이 크므로 견적 시점 기준 재확인 권장.

샘플 / 실물 확인

  • 벽지 · 알판 · 가구 마감재는 모니터·인쇄물과 실물 발색 차이가 있습니다. 최종 선정 전 실물 샘플 확인을 권장드립니다.
  • 당사 자료실 / 협력사 쇼룸 동행 방문 안내 가능.
