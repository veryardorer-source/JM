# 📖 JM 관리 시스템(JM OS) 사용 설명서 — 대표님용

> 쉽게 쓴 사용법입니다. 2026-06-22 기준.
> ※ 실제 화면의 메뉴 이름은 조금 다를 수 있어요. 다음에 앱 화면을 같이 보며 정확히 맞추겠습니다.

## 1. 이게 뭔가요
회사의 **현장(공사) 관리 프로그램**입니다. 현장별 일정·사진·비용·직원 업무·영수증·공지 등을 한곳에서 봅니다. 노션 대신 쓰는 우리 회사 전용 시스템이에요.

- **접속 주소**: https://jm-interior.vercel.app
- 컴퓨터·휴대폰 브라우저(크롬 등)에서 주소만 치면 들어갑니다. (설치 필요 없음)

## 2. 로그인하기
1. 위 주소 접속
2. 이메일 + 비밀번호 입력 → 로그인
3. 처음 쓰는 직원은 **회원가입** 먼저 (이메일/비밀번호/이름)

## 3. 직원 가입시키기 (지금 방식)
1. 직원에게 주소를 알려주고 **회원가입**하게 합니다 (이메일·비밀번호·이름).
2. 가입하면 **바로 로그인**됩니다. (이메일 인증 절차 꺼둠)
3. **처음엔 모두 '현장팀(field)' 권한**으로 시작합니다.
4. 더 높은 권한이 필요한 직원은 **대표님이 역할을 올려줘야** 합니다.
   - ⚠️ 지금은 이 "역할 바꾸기"가 앱 안에 화면이 없어서, 필요하면 알려주세요(제가 처리). **곧 앱 안에서 직접 승인·권한 지정하는 기능을 만들 예정**입니다.

> 직원 가입이 다 끝나면, 외부인이 함부로 가입 못 하도록 "신규 가입 허용"을 꺼두는 걸 추천합니다(요청 시 처리).

## 4. 권한(역할) 3가지
| 역할 | 누구 | 할 수 있는 것 |
|------|------|----------------|
| **관리자(admin)** | 대표·이사 | 전부. 재무, 직원·권한 관리, 출금 승인 |
| **디자인(designer)** | 디자인팀 | 현장·자료·일정 관리, 영수증/출금 요청 |
| **현장(field)** | 현장팀 | 배정 현장 업무, 사진·영수증 올리기, 출금 요청 (재무·민감정보는 제한) |

## 5. 주요 메뉴 사용법
> 현장(프로젝트)을 중심으로 모든 게 연결됩니다.

- **현장 관리**: 공사 현장을 등록하고 상태(문의~완료)를 관리. 현장을 클릭하면 그 현장의 자료·일정·비용을 봅니다.
- **현장 배정**: 어떤 직원이 어떤 현장을 맡는지 지정.
- **일정/공정표**: 철거·목공·전기 등 공정 일정을 관리.
- **현장 자료**: 시공 전/후 사진, 도면, 문서 업로드.
- **영수증**: 직원이 영수증 사진과 금액을 올림 → 대표가 확인.
- **출금 요청**: 거래처 지급 요청을 올리고, 대표(관리자)가 승인.
- **공지사항**: 사내 공지 작성·확인.
- **회사 문서/매뉴얼**: 회사 규정·매뉴얼 보관.
- **재무**: 매출·고정비·급여·현장별 수익 (관리자 전용).

## 6. 대표님이 주기적으로 할 일
- 신규 가입 직원의 **역할 지정**(현장팀→디자인/관리자 등)
- **영수증 확인**, **출금 요청 승인**
- 공지사항 등록
- (직원 가입 끝난 뒤) "신규 가입 허용" 끄기

## 7. 자주 생기는 문제

**Q. 가입했는데 로그인하면 "이메일 또는 비밀번호가 올바르지 않다"고 나와요.**
- 과거 "이메일 인증 필요" 설정 때문에 생겼던 문제이고, 지금은 꺼서 해결됨.
- 그래도 안 되면 → 비밀번호를 잘못 입력했을 가능성. 비밀번호 재설정이 필요하면 알려주세요(제가 처리).

**Q. 직원 권한(역할)을 바꾸고 싶어요.**
- 현재는 알려주시면 제가 바꿔드립니다. (앱 내 권한 화면은 곧 추가 예정)

**Q. 외부 사람이 가입하면 어쩌죠?**
- 직원 가입이 끝나면 "신규 가입 허용"을 꺼두면 됩니다. 요청 주세요.
